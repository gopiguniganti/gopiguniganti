## 🖥️ HomeLab

Running **40+ Docker containers** on a self-hosted HP ProDesk — media automation, document management, photo backup, local AI, network monitoring, and more. All behind a reverse proxy with VPN-routed download clients.

→ **[gopiguniganti/HomeLab](https://github.com/gopiguniganti/HomeLab)** — full stack with compose files, configs, and setup docs

**Some highlights:**
- 📸 [Immich](https://immich.app/) — self-hosted Google Photos with ML face recognition
- 🤖 [Open WebUI](https://github.com/open-webui/open-webui) + local LLMs — fully private AI interface
- 📄 [Paperless-NGX](https://docs.paperless-ngx.com/) — OCR document pipeline
- 📊 [ntopng](https://www.ntop.org/) + [Scrutiny](https://github.com/AnalogJ/scrutiny) — network & disk monitoring
- ⚙️ [n8n](https://n8n.io/) — workflow automation
